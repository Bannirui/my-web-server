my-web-server
---
A WEB SERVER BASED ON SOCKET AND IO-SELECTOR

### OPTIONAL UTILS

- ab
  - INSTALL `sudo apt install apache2-utils`
  - VERIFY `ab -V`
  - TEST `ab -n 1000 -c 10 http://127.0.0.1:9527/`

- telnet
  - INSTALL `sudo apt install telnet`
  - TEST
    - `telnet www.baidu.com 80`
    - `GET / HTTP/1.1`
